Polars検索機能の説明

基本的な検索機能

Polarsデータフレームを使用した検索機能は、以下の特徴があります：

  • 複数列での検索（名前、部署、スキルなど）
  • 大文字小文字を区別しない検索（オプションで区別も可能）
  • 複数キーワードでの検索（AND/OR条件）
  • 数値範囲での検索（評価など）
  • フィルタリングとの組み合わせ（部署など）

実装例


# 基本的な検索
result = search_dataframe(df, '営業')

# 特定の列のみを検索
result = search_dataframe(df, 'Python', columns=['skills'])

# 複数キーワードでの検索（OR条件）
result = search_with_multiple_keywords(df, ['Python', '営業'])

# 複数キーワードでの検索（AND条件）
result = search_with_multiple_keywords(df, ['Python', '開発'], match_all=True)

# 高度なフィルタリングと検索の組み合わせ
result = advanced_search(df, query='Python', filters={'department': '開発部'})

# 数値範囲での検索
result = search_numeric_range(df, 'rating', min_value=4.0)
